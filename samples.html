Our Work

Data Center Inventory
A fully loaded Inventory tracking, mapping, and management solution for Physical and Virtual Hosts at Indiana University. This site also allows for bulk upload with excel spreadsheets that import directly into the system.

Site Manager
An SFTP based solution for publishing Dev web pages to Production Web Servers with version tracking, and user logging.

3SD Games web store
A fully multi-lingual and multi-currency shopping chart and inventory system, with capability to parse and collect manufacturer codes, inventory levels, and stock.

Firewall Viewer and Requests
An enterprise class tool at Indiana University to manage requests into the border firewall, scrape and parse the firewall settings into a display, and allow escalation and system notices.

Gaming@IU Event Registration and Voting
For the Gaming @ IU club, we wrote a event registration portal, built off the existing phpBB forum space, that used Paypal payments, game voting, sponsor registration and tracking for events.

Online Coursework and Troubleshooting Guide
For AC/C Tech, we developed an online course system for their instructors and students, as well as a fully functional "Choose your own Adventure" style troubleshooting guide to help with equipment repair and installation.
